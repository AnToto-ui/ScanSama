À propos du Projet

Bienvenue sur ma page de profil ! Je m'appelle Antoine Noémie, et je suis passionné par l'univers des mangas. Dans le cadre d'un cours de développement en HTML et CSS, j'ai entrepris un projet captivant : la création d'un site web dédié à la découverte des nouveaux scans de mangas récemment sortis.

Contexte du Projet

Le projet s'inspire d'un site existant nommé Anime Sama, qui est bien connu pour le streaming de séries animées japonaises. Mon objectif était d'étendre ce concept pour inclure les mangas, offrant ainsi une plateforme où les fans peuvent suivre les versions manga de leurs séries animées préférées.

Objectif et Réalisation

L'objectif principal de ce projet était de reproduire et d'améliorer l'expérience utilisateur d'Anime Sama, mais cette fois, en se concentrant sur les mangas. Voici les étapes clés et les fonctionnalités que j'ai implémentées :

1- Développement en HTML et CSS : En utilisant les langages HTML et CSS, j'ai créé les différentes pages du site, en m'assurant qu'elles soient responsives et accessibles sur différents appareils.
2- Catalogue de Scans de Manga : J'ai intégré une section dédiée où les utilisateurs peuvent découvrir les derniers scans de mangas, avec des descriptions et des images pour chaque titre.
3- Menu de Navigation : Dans le menu de mon site il y est inclut des boutons pour les sections Planning et Catalogue, Celles-ci sont reliés aux pages correspondantes du site Anime Sama, offrant ainsi une continuité et une facilité d'accès pour les utilisateurs.

Technologies Utilisées

Pour mener à bien ce projet, j'ai utilisé les technologies suivantes :

- Visual Studio Code comme environnement de développement intégré (IDE) pour écrire et organiser le code efficacement.
- HTML pour structurer le contenu du site.
- CSS pour styliser et rendre le site attrayant visuellement.
